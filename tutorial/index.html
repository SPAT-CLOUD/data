HarScript Tutorial

HarScript History
  1. HarScript is Developed at "A-17 Quarters,vital nagar,Godhavarikhani".

  2. Main purpose of this is to Genrate A Modal Alerts By Single function call.

  3. And this reduces some code by linking it's library.

  4. for example , DOM Based such as document.getElementById,document.getElementsByTagName Can be reduced without jQuery.


Modal Alert Functions
  1. There are mainly four Modal Alert functions

  2. HAlertBox(),HWarning(),HAlert(),HWindow()

  3. HAlertBox Has disadvantage Cant Show big message in it's innerBox.

  4. HWarning onloads it will vibrate for 2 seconds it Also Cant show big message.

  5. HAlert is iframe based Alert box it shows big message but you need to scroll.

  6. HWindow it can show big message but problem with it is it cant show large message and also close button width depends upon style of your site.

  7. NOTE:These Functions Are asynchronus functions (not immediate action)

  8. For outputs visit here:Output for Modal Functions

Instructions before you learn HarScript
  1. When You are dealing with Modal Alert Functions You need compulsory two Tags <body>,<head>.

  2. The External JavaScript File must be linked And set In Head Section As Mentioned in instruction 3 Shows How to Link.

  3. Example Structure:
    <!DOCTYPE html>
    <html>
    <head>
    <!--copy below script tag to run in your website-->
    <script async src="https://cdn.jsdelivr.net/gh/SPAT-CLOUD/data@main/HarScript/HarScript.v3.0.Latest.min.js"></script>
    </head>
    <body>
    </body>
    </html>

DOM Functions tutorial
  1. This Consists of basically 2 functions $HId() , $HTag()

  2. $HId() takes id of Element As Argument

  3. $HTag() takes Argument As TagName of element Ex:<p> TagName is Accessed as $HTag('p')

  4. First create A your_file_name.html file

  5. Now save this below code in your your_file_name.html file.

  6. Code for $HId():
    <!DOCTYPE html>
    <html>
    <head>
    <script async src="https://cdn.jsdelivr.net/gh/SPAT-CLOUD/data@main/HarScript/HarScript.v3.0.Latest.min.js"></script>

    </head>
    <body>
    <p id='o'>Test</p>
    </body>
    <script>
    $HId('o').innerHTML = "anything";
    </script>
    </html>


  7. Code for $HTag():
    <!DOCTYPE html>
    <html>
    <head>
    <script async src="https://cdn.jsdelivr.net/gh/SPAT-CLOUD/data@main/HarScript/HarScript.v3.0.Latest.min.js"></script>

    </head>
    <body>
    <p>Test</p>
    </body>
    <script>
    $HTag('p')[0].innerHTML = "anything";
    </script>
    </html>
